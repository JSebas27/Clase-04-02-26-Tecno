# Clase-04-02-26-Tecno
Instrucciones para ejecutar el codigo:
`### 1.` 1- Clonar el repositorio
2- Abrir VS Code
3. Asegurarse de activar entorno virtual:
Comando 1: python -m venv venv
Comando 2: venv\Scripts\activate
4. Instalar dependencias 
Comando 3: pip install -r requirements.txt
5. Ejecutar to do list:
Comando 4: python to-do-list-app/app.py
6. Ejecutar flask:
Comando 5: python flask-app/app.py
7. Ejecutar la aplicacion
Comando 6: python .\app.py
8. Abrir URL de la aplicacion dando ctrl+clic y agregarle al final del link /inicio.
